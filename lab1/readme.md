**Група:** КВ-31

### Тема та завдання

**Тема:** Розробка статичного інтерфейсу Web-додатка.  
**Тематика додатку:** Облік робочого часу (SessionTracker).

**Мета роботи:** ознайомитись із засобами мов HTML5 та CSS3 та навчитись застосовувати бібліотеки (в даному випадку Tailwind CSS) щодо швидкої побудови Web-інтерфейсу користувача.

**Загальне завдання:** розробити статичні Web-сторінки додатку та перевірити коректне їх відображення на екранах різного розміру.
